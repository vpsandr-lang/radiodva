# 🎧 RADIO DVA — Двойная Волна

**RADIO DVA** — это онлайн-радиостанция с уникальной концепцией: **50% топ-хитов России + 50% мировых бестселлеров**. Ведущий он же диджей — живой голос, авторские миксы, никаких роботов.

## 🚀 Быстрый старт

```bash
# Просто открой index.html в браузере
open index.html

# Или запусти локальный сервер
python3 -m http.server 8080
# → http://localhost:8080
```

## 🏗️ Структура проекта

```
/root/Radio/
├── index.html          # Главная страница с плеером
├── advert.html         # Страница для рекламодателей
├── css/
│   └── style.css       # Тёмная тема, анимации, responsive
├── js/
│   ├── data.js         # Плейлист и конфигурация стрима
│   ├── player.js       # Класс RadioPlayer (Audio)
│   └── main.js         # UI-контроллер, анимации, интеграция
├── MONETIZATION.md     # Стратегия монетизации и продажи
└── README.md           # Этот файл
```

## 🔧 Настройка

1. Открой `js/data.js`
2. Замени `streamUrl` на URL твоего Icecast/SHOUTcast стрима
3. Настрой треки в `TRACK_HISTORY` и `CURRENT_TRACK`
4. Обнови контакты в `index.html`

## 🌐 Для продакшена

### Стриминг-сервер (выбери один):
- **Icecast 2** (бесплатно) — `apt install icecast2`
- **SHOUTcast** (платно)
- **WHMSonic** (если на shared хостинге)
- **Аренда радио-сервера** — от 500 ₽/мес

### Хостинг сайта:
- **VPS** (Timeweb, FirstVDS, Beget) — от 300 ₽/мес
- **Netlify / Vercel** — бесплатно (только статика)
- **GitHub Pages** — бесплатно

### Домен:
- `radiodva.ru` — ~200 ₽/год
